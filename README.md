# 📊 Blinkit Sales Performance Dashboard

## 📌 Project Overview

This project presents an interactive Power BI dashboard built to analyze Blinkit's sales performance, customer behavior, payment preferences, and delivery metrics. The dashboard transforms raw sales data into meaningful business insights that can help improve decision-making.

The project demonstrates skills in data cleaning, data modeling, DAX calculations, data visualization, and business intelligence using Power BI.

---

## 🎯 Objectives

- Analyze overall sales performance
- Track monthly revenue trends
- Compare sales across different cities
- Identify top-performing product categories
- Analyze customer ratings
- Understand payment method preferences
- Monitor delivery performance and returned orders

---

## 🛠️ Tools & Technologies

- Power BI
- Power Query
- DAX
- Microsoft Excel / CSV Dataset

---

## 🧹 Data Cleaning & Preprocessing

Before building the dashboard, the dataset was cleaned and prepared using Power Query.

The following preprocessing steps were performed:

- Checked the dataset for missing values.
- Filled missing values in the **City** column with **"Unknown"**.
- Filled missing values in the **Payment Method** column with **"Unknown"**.
- Verified and corrected data types (Date, Number, and Text).
- Removed duplicate records (if any).
- Renamed column headers for better readability.
- Ensured consistency across categorical values.
- Prepared the cleaned dataset for analysis and visualization.

The cleaned dataset was then loaded into Power BI for data modeling and dashboard development.

## 🔍 Key Insights

- Hyderabad generated the highest sales among all cities.
- Grocery was the top-performing product category.
- UPI was the most preferred payment method.
- Monthly revenue showed fluctuations throughout the year.
- Most orders were successfully delivered with a low return rate.
- Customer ratings remained positive across most product categories.

## ✅ Conclusion

This dashboard provides a comprehensive view of Blinkit's sales and customer performance. It helps identify revenue trends, top-performing cities, popular product categories, customer preferences, and delivery efficiency.

The insights generated from this dashboard can support better business decisions related to inventory planning, marketing strategies, customer experience, and operational improvements.
